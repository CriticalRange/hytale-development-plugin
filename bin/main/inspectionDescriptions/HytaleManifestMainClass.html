Validates that the Main class specified in manifest.json is valid.

This inspection checks that:

  • The specified class exists in the project
  • The class extends PluginBase or JavaPlugin
  • The class is not abstract
  • The class has a valid constructor (accepts JavaPluginInit parameter)

Example of valid manifest.json:

{
    "Main": "com.example.MyPlugin"
}

Where MyPlugin extends JavaPlugin and has the required constructor.
