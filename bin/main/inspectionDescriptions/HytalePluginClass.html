Validates that Hytale plugin classes are properly structured.

This inspection checks plugin classes that extend PluginBase or JavaPlugin:

  • The class should not be abstract
  • The class must have a constructor that accepts JavaPluginInit parameter
  • The class should not be private

Example of valid plugin class:

public class MyPlugin extends JavaPlugin {
    public MyPlugin(JavaPluginInit init) {
        super(init);
    }
}
