Validates that all required fields are present and properly formatted in manifest.json.

Required fields:

  • Group - Package name format (lowercase, dot-separated, e.g., com.example)
  • Name - Plugin name (alphanumeric, may include dashes/underscores)
  • Version - Semantic version (e.g., 1.0.0 or 1.0.0-beta.1)
  • Main - Fully qualified class name of the plugin entry point

Example of valid manifest.json:

{
    "Group": "com.example",
    "Name": "MyPlugin",
    "Version": "1.0.0",
    "Main": "com.example.MyPlugin"
}
